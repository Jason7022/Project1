The executable version of the compiler is built for:
- Windows OS
- Google Chrome as the expected web browser to display the generated HTML output.

To run the compiler from the command line:
1. Open a terminal in the project directory.
2. Run:  lolmarkdownn.exe <inputfile.lol>
Example:  lolmarkdownn.exe test\Test1.lol
